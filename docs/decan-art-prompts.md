# TarotByte — The Decan Engine: 36 Canva Art Prompts

These are for Canva's **AI image generator (Magic Media → Images)**, or any text-to-image tool. Each of the 36 decans has a name (your IP), a zodiac sign, a decan number, a ruling planet, and a punchy visual prompt.

---

## HOW TO USE THIS (read first — 2 min)

### 1. Lock the "house style" so all 36 match
Cohesion is everything for a deck. **Paste this STYLE BLOCK at the START of every single prompt**, then add the card-specific line after it. Do NOT change the style block between cards.

> **STYLE BLOCK (copy every time):**
> "Tarot card illustration, vertical 2:3 portrait, ornate art-nouveau border with fine brass filigree and small celestial symbols, deep cosmic navy-and-black background with faint constellations and gold star-dust, luminous accents in antique gold and arcane cyan, painterly digital illustration with subtle steampunk-astrolabe detailing, symbolic and mystical, elegant and high-end, no text, no lettering, no words, no numbers."

Then add: `SCENE: <the card-specific line below>`

### 2. Settings in Canva
- **Aspect ratio:** choose **2:3 (portrait)** — standard tarot proportion.
- **Style:** pick a painterly / illustration style (avoid "photo").
- Generate **4 variations**, pick the best, regenerate any that drift.

### 3. Keep a color anchor
If a card comes out with the wrong palette, add to the end: *"strict palette: cosmic navy, black, antique gold, arcane cyan only."*

### 4. Naming your files
Save each as `decan-<sign>-<num>-<slug>.png` — e.g. `decan-scorpio-2-golden-return.png`. That matches how we'll wire them into the site.

### 5. IP note
The **names are yours** (original TarotByte IP). The underlying pip→decan mapping is the public-domain Golden Dawn system, so you're safe. Keep the art original (AI-generated from your prompts = fine).

## Production tip
Do them in **suit batches** (all Wands, then all Cups…) in one Canva session — the model stays more consistent within a session, and the four suits naturally cluster by palette (fire=warm gold/red, water=cool blue/teal, air=pale/violet, earth=amber/green over your navy base).
